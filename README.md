# Tic-Tac-Toe

# Initial setup
Install gems using:

` $ bundle install`

Initialize Rspec with: 

`$ bundle exec rspec --init`

# Run the tests
To run all the tests at once use:

`$ make test`

To run tests for individual components

`$ rspec ./spec/board_spec.rb`

or

`$ rspec ./spec/player_spec.rb`

# Run the game
To run the entire game use:

`$ make `
